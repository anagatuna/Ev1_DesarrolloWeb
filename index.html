Introducción general al Desarrollo Web

El desarrollo web es el proceso de crear, estructurar y mantener sitios y aplicaciones que funcionan en navegadores. Combina diversas técnicas, lenguajes y buenas prácticas para ofrecer experiencias digitales útiles, accesibles y atractivas.

Este proceso se organiza en tres capas principales:

  • Estructura (HTML): define el contenido y la organización de la página.
  • Presentación (CSS): controla el diseño, los colores y los estilos visuales.
  • Comportamiento (JavaScript): añade dinamismo e interactividad.

En esta evidencia nos enfocaremos principalmente en HTML y CSS, ya que constituyen la base de todo proyecto web y fueron los temas abordados durante estas semanas en clase.

Características del desarrollo web

Accesible

Un sitio web accesible garantiza que todas las personas, incluyendo aquellas con discapacidades, puedan navegar y entender el contenido. Esto incluye:

  • Texto alternativo en imágenes: <img alt="Descripción">
  • Navegación mediante teclado y lectores de pantalla
  • Contraste adecuado entre texto y fondo

La accesibilidad mejora la experiencia de usuario y cumple con buenas prácticas internacionales.

Responsivo

Un sitio responsivo se adapta a diferentes tamaños de pantalla, desde móviles hasta monitores grandes. Esto se logra mediante:

  • Diseño flexible con CSS y flexbox o grid
  • Uso de media queries para ajustar estilos según la resolución
  • Imágenes y contenido que se redimensionan correctamente

Así, los usuarios disfrutan de una experiencia consistente sin importar el dispositivo.

Semántico

HTML5 proporciona etiquetas semánticas que describen el propósito del contenido, ayudando a navegadores, motores de búsqueda y lectores de pantalla a entender la estructura de la página:

  • <header>: Encabezado principal
  • <nav>: Menú de navegación
  • <main>: Contenido principal
  • <section>: Secciones de contenido relacionadas
  • <article>: Artículos o bloques independientes
  • <aside>: Información secundaria o complementaria
  • <footer>: Pie de página

Usar HTML semántico mejora la accesibilidad, el SEO y la mantenibilidad del sitio.
