Introducción general al Desarrollo Web

El desarrollo web es el proceso de crear, estructurar y mantener sitios y aplicaciones que funcionan en navegadores. Combina diversas técnicas, lenguajes y buenas prácticas para ofrecer experiencias digitales útiles, accesibles y atractivas.

Este proceso se organiza en tres capas principales:

  • Estructura (HTML): define el contenido y la organización de la página.
  • Presentación (CSS): controla el diseño, los colores y los estilos visuales.
  • Comportamiento (JavaScript): añade dinamismo e interactividad.

En esta evidencia nos enfocaremos principalmente en HTML y CSS, ya que constituyen la base de todo proyecto web y fueron los temas abordados durante estas semanas en clase.

Características del desarrollo web

Accesible

Un sitio web accesible garantiza que todas las personas, incluyendo aquellas con discapacidades, puedan navegar y entender el contenido. Esto incluye:

  • Texto alternativo en imágenes: <img alt="Descripción">
  • Navegación mediante teclado y lectores de pantalla
  • Contraste adecuado entre texto y fondo

La accesibilidad mejora la experiencia de usuario y cumple con buenas prácticas internacionales.

Responsivo

Un sitio responsivo se adapta a diferentes tamaños de pantalla, desde móviles hasta monitores grandes. Esto se logra mediante:

  • Diseño flexible con CSS y flexbox o grid
  • Uso de media queries para ajustar estilos según la resolución
  • Imágenes y contenido que se redimensionan correctamente

Así, los usuarios disfrutan de una experiencia consistente sin importar el dispositivo.

Semántico

HTML5 proporciona etiquetas semánticas que describen el propósito del contenido, ayudando a navegadores, motores de búsqueda y lectores de pantalla a entender la estructura de la página:

  • <header>: Encabezado principal
  • <nav>: Menú de navegación
  • <main>: Contenido principal
  • <section>: Secciones de contenido relacionadas
  • <article>: Artículos o bloques independientes
  • <aside>: Información secundaria o complementaria
  • <footer>: Pie de página

Usar HTML semántico mejora la accesibilidad, el SEO y la mantenibilidad del sitio.

Elementos fundamentales

El desarrollo web no solo se trata de escribir código, sino de combinar varios elementos para crear sitios funcionales, atractivos y accesibles para los usuarios en internet. Entre los elementos fundamentales del desarrollo tenemos los siguientes:

Estructura

  • Se refiere a cómo se organiza la información en la página. Una estructura bien organizada facilita que los usuarios y los motores de búsqueda comprendan el contenido.

  • HTML es el principal lenguaje para crear la estructura, definiendo encabezados, párrafos, tablas, secciones, etc.

Estilo y Diseño

  • Un buen diseño no solo hace la página atractiva, sino que mejora la experiencia de usuario y la navegación.

  • CSS se encarga de la apariencia visual de los elementos HTML. Permite definir colores, tipos de letras, tamaños, márgenes, alineaciones, fondos, etc.

Accesibilidad

  • Consiste en hacer que los sitios web sean utilizables, intuitivas y amigables para todas las personas.

  • Incluye el uso de etiquetas, texto alternativo en imágenes, entre otros.

Diseño Adaptativo

  • Permite que las páginas se ajusten a diferentes tamaños de pantalla y dispositivos, desde computadoras de escritorio, laptops, tablets y teléfonos móviles.

  • Garantiza que el contenido sea legible y fácil de interactuar sin importar el dispositivo que se use.

Optimización y Rendimiento

  • Incluye técnicas para que las páginas carguen rápido y funcionen correctamente.

Aplicaciones del desarrollo web

Las aplicaciones de desarrollo web son muy diversas y están presentes en nuestra vida cotidiana, en la educación, los negocios y la comunicación digital.
Entre las más destacadas:

Sitios Informativos y Blogs

  • Permiten compartir noticias, investigaciones, contenido educativos. Todos estos lo podemos encontrar mediante blogs, páginas de noticias, portales de información, etc.

Tiendas en Línea y Comercio Electrónico

  • Facilitan la venta de productos y servicios de manera digital.

  • Incluyen carritos de compra, inventarios y seguimiento de pedidos.

Redes Sociales

  • Permiten a las persones interactuar y mantenerse comunicados unos con otros. Ejemplos muy sencillos de estas pueden ser: Facebook, Instagram, X, etc.

Aplicaciones Web para Empresas y Servicios

  • Automatizan procesos internos y externos, como gestión de clientes, contabilidad, reservas y seguimiento de proyectos.

  • Permiten optimizar recursos y mejorar la eficiencia operativa.

Educación y Aprendizaje En Línea

  • Plataformas educativas permiten cursos interactivos, evaluaciones, seguimiento de progreso y materiales multimedia.

  • Ejemplo: aulas virtuales, plataformas de e-learning como SuperProf, entre otras.

Entretenimiento y Medios Digitales

  • Proporcionan experiencias interactivas y acceso a contenido multimedia de manera inmediata.

  • Ejemplo de estos sitios pueden ser los destinados a música, video, juegos y transmisiones en vivo. Por ejemplo Youtube, Spotify, Twitch, etc.
