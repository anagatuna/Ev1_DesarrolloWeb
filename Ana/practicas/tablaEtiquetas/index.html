Código, subíndices y superíndices
<code> Permite escribir código. print("Hola Mundo")
<sub> Subíndice, se utiliza para escribir texto en una posisión inferior a la línea base. H2O
<sup> Superíndice, se utiliza para escribir texto en una posisión superior a la línea base. X2
